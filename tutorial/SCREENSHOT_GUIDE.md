# Screenshot Guide for CardLister Pro Tutorial

Take these 5 screenshots and save them in the `screenshots` folder with the exact filenames below.

## Quick Steps

1. Run the app: `python app.py` → open http://localhost:5001
2. Use **Win + Shift + S** (Windows) or **Snipping Tool** to capture
3. Save each screenshot with the exact filename in `tutorial/screenshots/`
4. Open `TUTORIAL_WITH_SCREENSHOTS.html` in your browser

---

## Screenshot 1: `01-landing.png`
**What to capture:** Full landing page
- Show the hero section ("List Premium Sports Cards on eBay in Seconds")
- "Login" button (top-right)
- "Start Now - $14.99/mo" and "View Plans - 50% OFF" buttons
- Pricing cards at bottom (optional)

## Screenshot 2: `02-listing-details.png`
**What to capture:** After login – Step 1 "Listing Details"
- Set Name input field
- Description textarea
- Image URL field
- Default Price, Default Quantity (if visible)
- Scroll so the full Step 1 card is visible

## Screenshot 3: `03-fetch-checklist.png`
**What to capture:** Step 2 "Fetch Checklist"
- Checklist URL input (can paste a Beckett URL)
- **Type** dropdown (Base Cards, Insert Cards, etc.)

- Default Price/Qty fields

## Screenshot 4: `04-cards-loaded.png`
**What to capture:** After fetching – Step 3 "Your Cards"
- Search bar ("Search by card number or player name...")
- Card table with columns: #, Player Name, Team, Price, Qty
- At least a few rows of cards visible
- Buttons: Select All, Bulk Edit, Export CSV, etc.

## Screenshot 5: `05-verify-publish.png`

- Verify Draft button
