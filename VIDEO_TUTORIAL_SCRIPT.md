# CardLister Pro - Step-by-Step Video Tutorial Script

Use this script when recording your tutorial video. Pause at each step to show the screen, then perform the action.

---

## INTRO (5–10 seconds)
- **Say:** "Welcome to CardLister Pro – the tool for listing sports cards on eBay quickly."
- **Show:** Landing page with pricing and features.

---

## STEP 1: LOGIN (15–20 seconds)

1. **Show:** Landing page
2. **Click:** "Login" button (top-right or "Start Now - $14.99/mo")
3. **Say:** "Sign in with your eBay account."
4. **Show:** Login page
5. **Click:** Login with your credentials (or show mock login flow)
6. **Show:** Redirect to main app

---

## STEP 2: SET UP LISTING DETAILS (30–45 seconds)

1. **Show:** App screen – Step 1 section "Listing Details"
2. **Say:** "In Step 1 we set the basic listing info."

3. **Click in Set Name field**
   - **Say:** "Enter your set name, like '2024 Topps Chrome Basketball Base.'"

4. **Click in Description field**
   - **Say:** "Add your description. You can also auto-fill this later from the checklist."

5. **Click in Image URL field**
   - **Say:** "Paste your main product image URL – usually from your hosting or eBay."

6. **Optional:** Show Category, Condition, etc. if visible

---

## STEP 3: FETCH CHECKLIST (45–60 seconds)

1. **Scroll to Step 2 – "Fetch Checklist"**
2. **Say:** "Step 2 is where we pull the card checklist from Beckett or Cardsmiths."

3. **Click in Checklist URL field**
   - **Say:** "Paste the full Beckett checklist URL – for example, the 2024-25 Topps Chrome Basketball checklist."
   - **Paste:** `https://www.beckett.com/news/2024-25-topps-chrome-basketball-cards/`

4. **Click the Type dropdown**
   - **Say:** "Choose what type of cards to fetch."
   - **Options:** Base Cards, Insert Cards, Autographs, #'ed, Parallels
   - **Select:** "Base Cards" for this example

5. **Click "Fetch Checklist" button**
   - **Say:** "Click Fetch Checklist. The bot will scrape the page and load all cards."
   - **Wait:** For the loading to finish

6. **Show:** Cards populated in the table
   - **Say:** "You can see all 200 base cards loaded. The description is often auto-filled too."

7. **Optional:** Point out the **Search bar** above the cards
   - **Say:** "Use the search bar to quickly find a card by number or player name."

---

## STEP 4: REVIEW & EDIT CARDS (30–45 seconds)

1. **Show:** The card table (Your Cards section)
2. **Say:** "Review the cards. Each row has number, name, team, price, and quantity."

3. **Click a Price field**
   - **Say:** "Set a default price, or type prices for each card."

4. **Click a Quantity field**
   - **Say:** "Enter quantity. Only cards with quantity greater than zero are listed."

5. **Optional:** Use **Bulk Edit**
   - **Click:** "Bulk Edit" button
   - **Say:** "Select cards with the checkbox, then use Bulk Edit to set price and quantity for many cards at once."

6. **Optional:** Show **Search**
   - **Type:** A player name or card number
   - **Say:** "The search bar filters the list as you type."

---

## STEP 5: VERIFY & PUBLISH (30–45 seconds)

1. **Scroll to Step 4 – "Verify & Publish"**
2. **Say:** "When everything looks good, move to Step 4."

3. **Click "Verify Draft" or similar**
   - **Say:** "Verify checks that eBay will accept the listing."
   - **Show:** Success or any errors

4. **Click "Publish to eBay"**
   - **Say:** "Click Publish to send the variation listing to eBay."
   - **Show:** Confirmation or success message

---

## QUICK TIPS (15–20 seconds)

- **Say:** "A few tips:"
  - "Use the parallel type dropdown when you fetch Parallels or #'ed cards."
  - "Export to CSV to backup or edit in a spreadsheet, then import back."
  - "Validate cards to catch missing prices or quantities."

---

## OUTRO (5 seconds)

- **Say:** "That’s CardLister Pro – list your sports cards on eBay in minutes. Thanks for watching."

---

## RECORDING CHECKLIST

- [ ] Clear browser cache for a clean UI

- [ ] Slow mouse movement so viewers can follow clicks
- [ ] Pause briefly after each important click
- [ ] Show both success and possible error messages
- [ ] Keep total video under 5–6 minutes for a concise tutorial

- **Mac:** QuickTime, OBS Studio, or built-in screen recording
